1.1    'pure' clojurescript hashmaps

Remember, these are tests in clojurescript!

(def o {:a 1 :b 2 :c 3})

(o :a)

=> 1

(o :a)

=> #(not= 0 %)

(get o :b)

=> 2

(:c o)

=> 3

(select-keys o [:a :b])

=> {:a 1 :b 2}
